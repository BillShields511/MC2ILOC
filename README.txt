# MC2ILOC
Mini-C to ILOC (Intermediate Language for Optimizing Compilers) Compiler

Please structure the contents of this git repository and the contents of nickle as follows.

MC2ILOC:
|------> nickle/
|------> partX.Y/

nickle can be found at: https://github.com/drz64/nickle.git

An additional README detailing mini-c to ILOC can be found within the partX.Y/ 
subdirectory, where x and y represent the current version of the project.
